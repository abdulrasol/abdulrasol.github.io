SWEEP: Reclaim Your Workspace

A professional command center to clean hidden developer artifacts. Built with Rust, Tauri v2, and React. Cross-platform for macOS, Windows, and Linux.

macOS Windows Linux

SWEEP Features

🧹

Deep Multi-Environment Clean

Cleans Flutter, Android (AVD/Gradle), Xcode (DerivedData), Node.js, Rust (Cargo), Python (Conda/Pip), Docker, Unity, Unreal Engine, and Adobe caches.

📊

Real-Time Dashboard

Live CPU, RAM, and disk space monitoring in a premium glassmorphism UI.

🔍

Smart Target Scan

Select your development directory and let SWEEP intelligently find garbage artifacts automatically.

🛡️

Safety First

Review every file before deletion. No accidental data loss. Permanent deletions are clearly flagged.

🖥️

Cross-Platform

Native builds for macOS (.dmg), Windows (.msi), and Linux (.deb) using Tauri v2 and Rust.
