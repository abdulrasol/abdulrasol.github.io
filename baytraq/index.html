Core Operations

العمليات الأساسية

🛒

Point of Sale (POS)

نقطة البيع

Bilingual interface optimized for quick actions with integrated camera/QR scanning, collapsible scanner feed to avoid layout overflows, and express multi-cart system.

واجهة ثنائية اللغة محسّنة للإجراءات السريعة مع مسح الباركود/QR المدمج، ونافذة كاميرا مرنة لتجنب التداخل، وسلة مشتريات متعددة وسريعة.

📦

Inventory Management

إدارة المخزون

Real-time stock tracking with auto-deductions on sales, color-coded low stock alerts, support for physical products & service items, and supplier/vendor linking.

تتبع المخزون بالوقت الفعلي مع خصم تلقائي للمبيعات، وتنبيهات انخفاض المخزون بالألوان، ودعم المنتجات والخدمات، وربط الموردين.

📄

Advanced Invoicing

فواتير متقدمة

Create custom invoices with item/invoice-level discounts, track paid/unpaid/partial states, view mutation history, and print formatted PDFs with store logos.

إنشاء فواتير مخصصة بخصومات متنوعة، وتتبع حالة الدفع (مدفوعة، غير مدفوعة، جزئية)، والاطلاع على سجل التعديلات، وطباعة PDF مع شعار المتجر.

Financials & Analytics

المالية والتحليلات

📈

Insights Dashboard

لوحة تحكم التحليلات

Real-time metrics showing Total Sales, Net/Gross Profits, Expenses, and Debts with multi-period filters (Week, Month, Year) and top-sellers charts.

مؤشرات مالية فورية للمبيعات، صافي وإجمالي الأرباح، المصروفات، والديون مع فلاتر للفترات الزمنية ورسوم بيانية لأكثر المنتجات مبيعاً.

💸

Expense Tracking

تتبع المصروفات

Log one-time expenses (Rent, Logistics, Salaries) and set up recurring expense templates with automated due calculation and alerts.

تسجيل المصروفات لمرة واحدة (إيجار، لوجستيات، رواتب) وإعداد قوالب مصروفات دورية مع احتساب تلقائي للمستحقات والتنبيهات.

🏛️

Debt & Credit Management

الديون والائتمان

Running customer ledger balances, automatic debit/credit posting on invoices, and a self-healing database engine that auto-repairs sync anomalies.

حسابات جارية لأرصدة العملاء، وتسجيل تلقائي للديون والائتمان بالفواتير، ومحرك قاعدة بيانات يقوم تلقائياً بإصلاح اختلافات المزامنة.

Networking, Sync & Multi-Device

المزامنة وتعدد الأجهزة

🌍

Real-Time Cloud Sync

المزامنة السحابية

Seamless multi-tenant synchronization powered by Cloud Firestore with secure custom rules and anonymous authentication fallbacks.

مزامنة سحابية سلسة ومتعددة المتاجر مدعومة بقواعد Firestore الآمنة وتسجيل الدخول المجهول الاحتياطي لحماية بياناتك.

📶

Local LAN Sync (Offline-First)

المزامنة المحلية بدون إنترنت

Master-Slave LAN architecture over Wi-Fi using shelf WebSockets, Drift SQLite local storage, and Last-Write-Wins (LWW) conflict resolution.

هيكلية رئيسي/تابِع للشبكة المحلية عبر Wi-Fi باستخدام WebSockets، وتخزين SQLite محلي مع معالجة تعارض المزامنة بالطوابع الزمنية.

🔍

Price Checker (Kiosk Mode)

قارئ الأسعار

Dedicated station for customer membership kiosk scanning, with offline fallback mode querying local replica database if connection drops.

محطة مخصصة للعملاء للبحث ومسح المنتجات لمعرفة الأسعار، مع وضع عدم الاتصال الذي يستعلم من نسخة قاعدة البيانات المحلية عند انقطاع الشبكة.

Identity & Access Control

الهوية والصلاحيات

👥

Staff Roles (RBAC)

إدارة الموظفين والصلاحيات

Role-Based Access Control (Admin, Manager, Cashier, Staff, PriceViewer) with screen guarding and secure PIN or QR-code logins.

التحكم في الوصول بناءً على الأدوار (مدير، محاسب، كاشير، موظف) مع حماية الشاشات وتسجيل دخول آمن برمز PIN أو رمز الاستجابة السريعة.

System Utilities

أدوات النظام

🛠️

Utility & Customization Suite

حزمة الأدوات والتخصيص

JSON Backup/Restore, 12 built-in languages, custom branding (company details, decimal limits, currency symbols), and fully responsive Light/Dark theme support.

نسخ احتياطي واستعادة بصيغة JSON، دعم 12 لغة مدمجة، تخصيص الهوية التجارية (تفاصيل المتجر، العملة، الكسور العشرية) ومظهر داكن/فاتح.
